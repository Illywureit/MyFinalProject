about the project

The project is part of the "data science" profession of the 10th grade students of the Alterman High School in Tel Aviv.

The project combines two main areas of knowledge:

* Programming in the Web Environment
* Data Science

As part of the project, I located databases on the network and intend to build around them a website that allows access to the databases, data analysis and presentation to users in a convenient way. The site will allow the user to explore the data in the chosen field and try to reach valuable insights and conclusions.

The project will be based on the following technological components:

* Python language.
* Pandas library.
* Flask infrastructure for building a web server.
